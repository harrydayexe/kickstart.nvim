## Install Instructions

 > Install requires Neovim 0.9+. Always review the code before installing a configuration.

Clone the repository and install the plugins:

```sh
git clone git@github.com:harrydayexe/kickstart.nvim ~/.config/harrydayexe/kickstart.nvim
NVIM_APPNAME=harrydayexe/kickstart.nvim/ nvim --headless +"Lazy! sync" +qa
```

Open Neovim with this config:

```sh
NVIM_APPNAME=harrydayexe/kickstart.nvim/ nvim
```
